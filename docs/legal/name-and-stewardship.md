# RepoGround name, origin, and stewardship record

## Recorded public use

The name **RepoGround** is documented as the canonical project and product
identity no later than 2026-07-17 in this public repository. Supporting
repository evidence includes:

- `docs/decisions/repoground-3-naming-and-migration.v1.json`;
- the public repository `heimgewebe/repoground`;
- the 3.0.0 changelog entry dated 2026-07-17;
- public Git history preserving the migration from earlier product names.

This record is evidence of project chronology. It is not a representation
that a trademark registration has been obtained or that no earlier third-
party rights exist.

## Defensive purpose

Name stewardship exists to keep users able to identify the official
project and to prevent impersonation, deceptive fundraising, and false
attribution. It is not intended to stop independent implementations,
forks, interoperability, criticism, or use of the Apache-licensed code.

## Custodial commitment

While Alexander Mohr remains the custodian, the RepoGround identity should:

1. not be sold or exclusively licensed to enclose the free project;
2. not be used to revoke or narrow Apache-2.0 rights;
3. be enforced proportionally and primarily against material confusion or
   deception;
4. move to a suitable non-profit community steward when a durable and
   accountable organization exists.

## Separation of rights

- `LICENSE` controls copyright and patent permissions for the software.
- `TRADEMARK_POLICY.md` controls potentially confusing use of the project
  identity.
- `FUNDING.md` records the intended voluntary and community-independent
  funding model.
